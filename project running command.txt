Local runbook
=============

Validate configuration:

    powershell -ExecutionPolicy Bypass -File scripts\ops\validate-env.ps1
    powershell -ExecutionPolicy Bypass -File scripts\ops\validate-route-map.ps1
    docker compose config --quiet

Build and test:

    .\mvnw.cmd -DskipTests compile -T 1C
    .\mvnw.cmd test -T 1C
    cd frontend
    npm run lint
    npm run build:next
    npm run test:e2e:smoke
    npm run test:e2e:visual

Start locally:

    docker compose up -d --build
    powershell -ExecutionPolicy Bypass -File scripts\ops\validate-compose-health.ps1

Inspect:

    docker compose ps
    docker compose logs -f gateway frontend foundation-service identity-service

Stop without deleting volumes:

    docker compose down

Only prune images, volumes, or git history when a human explicitly requests it.
