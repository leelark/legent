# Competitor Research

Fresh baseline date: 2026-05-20.

Store concise dated research notes here. File names should use:

`YYYY-MM-DD-<vendor-or-scan>-<area>.md`

Each note should include:

- `Source Register`: stable source ID, URL, access date, source type, vendor, and capability area.
- `Facts`: only what the source or current repository evidence proves.
- `Inference For Legent`: implications, confidence, and validation needed.
- `Gap Candidates`: gap ID, affected mode, severity, backlog ID or `PROPOSED:<slug>`.

Use source IDs like `SRC-SFMC-20260520-001` and reference those IDs from `docs/product/salesforce-parity-matrix.md`.

Allowed matrix status values:

- `Covered`
- `Partial`
- `Missing`
- `Evidence Required`
- `Blocked External`
- `No Action`

Do not copy long copyrighted content. Use links and short summaries.
