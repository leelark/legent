# Future Enhancement Roadmap

## Overall Product

- Add unified command center with cross-module tasks, alerts, and action recommendations.
- Add full OpenAPI generation and developer portal.
- Add audit timeline per campaign, subscriber, workflow, provider, and config.
- Add AI-assisted recommendations with explainable evidence and approval gates.
- Add multi-region and disaster recovery runbooks.

## Frontend UX

- Expand visual regression tests.
- Add guided tours for new operators.
- Add keyboard shortcuts for high-frequency workflows.
- Add saved views and advanced filters across tables.
- Add offline-friendly draft state for campaign/template builders.

## Identity And Foundation

- Add SCIM/SAML/OIDC enterprise identity.
- Add granular role templates and permission simulation.
- Add tenant-level policy packs and environment promotion workflows.

## Audience

- Add predictive segmentation, data quality scoring, and dedupe review UI.
- Add CSV mapping templates and import dry-run previews.
- Add consent source evidence and preference center customization.

## Content

- Add block-level approval, reusable design systems, and inbox preview providers.
- Add accessibility scoring for templates and landing pages.
- Add content localization and dynamic content preview matrix.

## Campaign

- Add campaign calendar, dependencies, blackout windows, and promotion workflow.
- Add stronger experiment winner analytics and auto-promotion guardrails.
- Add launch simulation with cost, provider capacity, and suppression impact.

## Delivery

- Add provider scoring model, adaptive throttling, and circuit breaker dashboard.
- Add replay simulation before execution.
- Add provider credential rotation workflow.

## Tracking And Analytics

- Add realtime attribution dashboards and cohort retention.
- Add anomaly detection for bounce/open/click patterns.
- Add exportable reports and scheduled reporting.

## Automation

- Add visual graph diff, approval gates, version rollback, and branch-level analytics.
- Add journey templates by lifecycle use case.
- Add dry-run dataset simulation.

## Deliverability

- Add DNS guided setup wizard with provider-specific instructions.
- Add BIMI, MTA-STS, TLS reporting, and blocklist monitoring.
- Add domain warmup plans and reputation forecasting.

## Platform And Infrastructure

- Add Helm chart or Kustomize overlays by environment.
- Add Terraform modules for cloud dependencies.
- Add centralized logging, distributed tracing, and SLO dashboards.
- Add CI/CD pipeline for build, test, image scan, deploy, smoke, rollback.
